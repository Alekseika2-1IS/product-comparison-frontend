---

## Описание проекта

Веб-приложение для наглядного сравнения характеристик 2–3 товаров.  
Реализован полный CRUD (создание, чтение, обновление, удаление) для товаров.  
Данные хранятся в `json-server` (имитация backend-API).

---

## Функциональные возможности

- **Каталог товаров** – карточки с изображением, названием, ценой, рейтингом и кнопками «В корзину», «Сравнить», «Ред./Удалить» (для админа).
- **Сравнение товаров** – выбор 2–3 товаров, умная таблица сравнения с подсветкой лучших/худших характеристик (зелёный – лучшее, красный – худшее, жёлтый – отличие).
- **Авторизация и роли** – регистрация, вход, разделение на обычных пользователей и администратора.
- **Корзина** – добавление товаров, изменение количества, оформление заказа (заказ сохраняется в localStorage и отображается в админ-панели).
- **Рейтинг и отзывы** – оценка товара звездами, написание отзыва (только для авторизованных), удаление отзывов администратором.
- **Админ-панель** – управление товарами (добавление, редактирование, удаление), просмотр заказов и пользователей.
- **Умное сравнение** – для экрана, процессора, памяти, камеры, аккумулятора, связи вычисляется интегральный балл; одинаковые строки скрываются.
- **Сохранение состояния** – выбранные для сравнения товары сохраняются в `localStorage` и не сбрасываются при обновлении страницы.

## Установка и запуск

### 1. Установите Node.js
Скачайте и установите Node.js с [официального сайта](https://nodejs.org/) (рекомендуется версия LTS).  
После установки проверьте, что всё работает: откройте терминал (командную строку) и выполните:

node --version
npm --version
2. Клонируйте репозиторий

git clone https://github.com/Alekseika2-1IS/product-comparison-frontend.git
cd product-comparison-frontend
3. Установите зависимости

npm install
4. Запустите сервер разработки (фронтенд)

npm run dev
После запуска приложение будет доступно по адресу: http://localhost:5173

5. Запустите json-server (имитация бэкенда)
Откройте новый терминал, перейдите в ту же папку и выполните:

npm run server
API будет доступно по адресу: http://localhost:3001/products

Использование
Авторизация
По умолчанию созданы:
Администратор: логин admin, пароль admin
Обычный пользователь: логин user, пароль user

Регистрация нового пользователя – автоматический вход после регистрации.

Корзина
Добавление товаров в корзину (требуется авторизация).

Просмотр корзины, изменение количества, удаление товаров, оформление заказа.

Рейтинг и отзывы
На карточке товара отображается средний рейтинг и количество оценок.

На странице товара можно поставить оценку и оставить отзыв.

Администратор может удалять любые отзывы (кнопка «Удалить» рядом с отзывом).

Сравнение товаров
Выберите от 2 до 3 товаров с помощью чекбоксов на карточках.

Нажмите кнопку «Сравнить».

На странице сравнения:

Одинаковые характеристики скрываются.

Числовые характеристики (память, камера, аккумулятор, частота экрана и т.д.) подсвечиваются: зелёным – лучшее значение, красным – худшее.

Операционная система: iOS – зелёный (лучший), Android – жёлтый (отличие).

Процессор: оценивается по встроенному рейтингу (Apple A19 Pro – лучший, MediaTek Helio – худший и т.д.).

Экран: оценивается по диагонали, разрешению, типу матрицы, частоте обновления, поддержке HDR.

Связь: оценивается по наличию 5G, eSIM, Wi‑Fi 6/7, Bluetooth 5.3+, NFC.

Админ-панель
Доступна только администратору (кнопка «Админка» в шапке).

Вкладка «Товары» – управление товарами (добавление, редактирование, удаление).

Вкладка «Заказы» – просмотр и подтверждение заказов (кнопка «Оформить заказ» удаляет заказ из списка).

Вкладка «Пользователи» – просмотр списка зарегистрированных пользователей.

Форма добавления/редактирования товара
Основные характеристики вынесены в отдельные поля:
Операционная система, Процессор, Экран, Память (ОЗУ/ПЗУ), Камера, Аккумулятор, Связь и подключение, Материал корпуса, Дополнительные функции.

Кнопка «+ Добавить характеристику» позволяет добавить любые дополнительные параметры (например, «ИК-порт», «Защита IP68»).
