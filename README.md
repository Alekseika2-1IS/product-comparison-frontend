# Интерфейс для сравнения товаров

Курсовой проект по дисциплине «Разработка кода информационной системы».  
**Тема:** Интерфейс для сравнения товаров  
**Студент:** Крылов Алексей  
**Специализация:** Frontend-разработка

---

## Описание проекта

Веб-приложение для наглядного сравнения характеристик 2–3 товаров.  
Реализован полный CRUD (создание, чтение, обновление, удаление) для товаров.  
Данные хранятся в `json-server` (имитация backend-API).

---

## Функциональные возможности

- Просмотр каталога товаров в виде карточек.
- Выбор от 2 до 3 товаров для сравнения.
- Страница сравнения с таблицей характеристик.
- Добавление нового товара через форму.
- Редактирование и удаление товаров.
- Визуальное выделение различий в характеристиках.
- Сохранение выбранных товаров в `localStorage`.
- **Авторизация и регистрация** (пользователи и администратор).
- **Корзина товаров** с возможностью оформления заказа.
- **Рейтинг и отзывы** к товарам (средняя оценка, отзывы, удаление отзывов администратором).
- **Админ-панель** для управления товарами, просмотра заказов и пользователей.

## Установка и запуск

### 1. Установите Node.js
Скачайте и установите Node.js с [официального сайта](https://nodejs.org/) (рекомендуется версия LTS).  
После установки проверьте, что всё работает: откройте терминал (командную строку) и выполните:

node --version
npm --version
Должны отобразиться версии (например, v20.18.0 и 10.8.0).

2. Клонируйте репозиторий (или скопируйте папку проекта)
Если у вас ещё нет проекта, склонируйте его с GitHub:


git clone https://github.com/Alekseika2-1IS/product-comparison-frontend.git
cd product-comparison-frontend
3. Установите зависимости

npm install
4. Запустите сервер разработки (фронтенд)

npm run dev
После запуска вы увидите сообщение:

VITE v...  ready in ... ms
➜  Local:   http://localhost:5173/
Не закрывайте это окно – в нём работает ваше приложение.

5. Запустите json-server (имитация бэкенда)
Откройте новый терминал, перейдите в ту же папку проекта и выполните:

npm run server
Должно появиться:

JSON Server started on PORT :3001
Endpoints: http://localhost:3001/products
6. Откройте приложение в браузере
Перейдите по адресу: http://localhost:5173

Использование
Авторизация
По умолчанию созданы:

Администратор: логин admin, пароль admin

Обычный пользователь: логин user, пароль user

Регистрация нового пользователя – автоматический вход после регистрации.

Корзина
Добавление товаров в корзину (требуется авторизация).

Просмотр корзины, изменение количества, удаление товаров, оформление заказа.

Рейтинг и отзывы
На карточке товара – средний рейтинг и количество оценок.

На странице товара – возможность поставить оценку и оставить отзыв.

Администратор может удалять любые отзывы.

Сравнение товаров
Выберите от 2 до 3 товаров с помощью чекбоксов на карточках.

Нажмите кнопку «Сравнить».

На странице сравнения:

Одинаковые характеристики скрываются.

Числовые характеристики подсвечиваются: лучшее – зелёным, худшее – красным.

Остальные отличия – жёлтым.

Админ-панель
Доступна только администратору (кнопка «Админка» в шапке).

Управление товарами (добавление, редактирование, удаление).

Просмотр списка заказов и пользователей.
